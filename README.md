# Progress 

Интерактивный круговой индикатор прогресса на чистом HTML, CSS и JavaScript.  
Позволяет показывать процент выполнения, вращать круг и скрывать/показывать его.  

---

## Особенности

- Плавная анимация прогресса.
- Вращение круга.
- Скрытие и показ круга с плавным эффектом.
- Контролируется через HTML-инпут и чекбоксы.
- Переиспользуемый JS-класс `ProgressCircle` с API для управления из кода.
- Легко интегрируется в любые проекты.
